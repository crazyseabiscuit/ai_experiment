【施工保】
Q1：为什么要施工保
1.	施工过程中的意外风险：
施工过程中随时可能遇到意外风险，工程损失不可控。
通过购买施工保险，可以在发生意外时获得赔偿，减轻经济损失。
2.	第三方维权意识提高：
施工涉及的第三方较多，三者维权意识提高，赔偿纠纷多。
购买施工保险可以减少因第三方维权导致的赔偿纠纷，保护施工方的合法权益。
3.	施工工人风险高：
施工工人风险高，自身无保险，一旦出事，施工单位赔偿责任大。
通过购买施工保险，可以为工人提供保障，减轻施工单位的赔偿责任。
4.	安监局要求：
安监局要求购买建工意外险等保险产品才允许进场施工。
这是合规要求，确保施工过程中的安全和风险管理。

Q2：平安施工保保障有哪些？
1. 工程自身：
自然灾害或意外事故造成的工程自身财产损失。
2.	三者责任：
意外事故造成周边第三者的人身伤亡或财产损失责任。
3.	员工意外：
施工人员因意外事故导致的人身伤亡造成的医疗费用支出。

Q3：平安施工保产品的亮点
1.	保障全面：
保工程、保三者、保人员，施工风险全覆盖。
2.	方案随心：
按工程造价收费，还有多种附加保障任你选。
3.	出单快捷：
按需勾选保障，支持二维码、APP出单，高效便捷。
这些亮点说明了施工保产品在保障范围、收费方案和操作便捷性方面的优势，旨在为用户提供全面、灵活且高效的保险服务。

Q4：平安施工保投保要求
1.	投保要求
工程造价：50万至1亿元
施工资质：在三级及以上
施工工期：不超过1年
2. 适用工程类型
2.1 道路类：
市区、县城道路新、改、扩建工程
路面铺设
2.2 管网类：
供水管网、燃气管网、污水处理厂
2.3 楼宇类：
医院、学校、办公楼、老旧小区改造、商场、幼儿园、图书馆、文化馆、博物馆、体育场馆、农贸市场
这些要求和适用范围为潜在投保人提供了明确的指导，确保他们能够根据自己的工程类型和资质选择合适的保险产品。

Q5：平安施工保方案有哪些？
保工程：保障由于自然灾害或意外事故造成的工程直接物质损失，保额等于工程造价（50万至1亿元）。
保三者：保障在施工过程中发生意外事故，造成第三者的人员伤亡或财产损失，具体保额以各机构保单为准。
保员工：
意外身故、残疾：保额可选10万、20万、30万、40万或50万。
意外伤害医疗：保额可选1万、2万、3万、4万或5万。
意外伤害住院津贴：50元/天或100元/天。

这些案例展示了施工保险在不同情况下提供的保障和理赔服务，帮助用户更好地理解保险的实际应用和价值。

Q7：是否任何类工程都可以投保？
回答：不是的。可投保的工程已在上述投保要求部分列明，除列明外的工程均不可保。常见的不可保类型包括：
o	纯拆除施工
o	涉海涉江或涉河工程
o	基坑支护工程
o	顶管工艺
o	暗挖工艺
o	边坡治理
o	任何植物
o	跨市跨区或跨县工程
o	城市河道清淤
o	桥梁工程
o	隧道工程
Q8: 工程保对比同类产品有哪些优势？
回答：出单快捷，统一条件线上投保，立等出单；保障全面，工程本身损失、周边三者损失，增加可选扩展费用损失、施工人员意外损失。
Q9: 投保资料有什么要求？
回答：施工资质在三级及以上的工程公司施工，拍照上传正式盖章的施工合同、营业执照两项资料即可。
Q10: 施工用机具、设备、机械装置等机器设备可以承保吗？
回答：不可承保。
